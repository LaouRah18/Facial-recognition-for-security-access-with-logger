# Facial-recognition-for-security-access-with-logger
This system is a facial recognition prototype designed for smart homes, offices and shops to grant or deny access  to guest. 

face_recognition_system/
│
├── save_face.py          → Capture & save the reference face (name.jpg)
├── recognize.py          → Recognize the face and greet the user
├── preprocess.py         → Image enhancement functions
├──utils.py 			   → Logger 
│
├── name.jpg                 → Saved/enhanced face sample
└── test.jpg              → Auto-generated test capture
